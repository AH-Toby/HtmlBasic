# JavaScript基础(二)

## 一、数据类型

### 1.1.简单的数据类型：

number： 数字类型，包括所有的整数、浮点数、特殊值

string：字符串类型，所有的字符串都是字符串类型

boolean：布尔类型，仅有两个值true,false

undefined：undefined类型，变量未定义时的值，这个值自己是一种类型

> 注：
>
> 1.数字类型里：不在区分整数、浮点数、Infinity、NaN、都是Number类型
>
> 2.null表示空对象，有的人也把null作为单独的数据类型

**变量数据类型：**

JavaScript是一种动态类型语言

变量的类型根据内部的数据而确定

但是变量内容是可以变动的，数据类型会根据内容的变化而变化

### 1.2.数据类型检测

**typeof**:检测数据类型

```js
typeof (变量名/字面量);
// 或者
typeof 变量名/字面量;
```

typeof本身就是一个关键字，后面的括号可写可不写

### 1.3.数据类型转换

**数字转字符串(数字+空字符串)：**

加号：在不同数据类型之间使用时，尤其时数字和字符串类型，加号表示连接符，将数字和字符串拼接，得到的结果时字符串

> 注意：
>
> 数字与数字类型，就是正常的运算符
>
> 在运算时候转换也称之为隐形转换

**字符串转数字**:

有两个函数 可以把非数值转换为数值：parseInt()和parseFloat()。

### 1.4.parseInt与parseFloat

**parseInt：**

作用：可以对数字进行取整

结构：

```js
parseInt(字符串，进制)
```

第一个参数：要转的字符串(可以省略前缀)

第二个进制：0，10以及默认值都是10，最多36位，如果指定了进制，那字符串前可以不带前缀"0"、"0o"、"0x"

**parseFloat：**

作用：将字符串转浮点数

结构：

```js
parseFloat(字符串)
```

就是将字符串转换为浮点数，直接省略第一个小数点后其他非数字的东西

**+:**

作用：可以将字符串转成数字，属于隐形转换，但是只能转换正确的数字，且进制必须带完整前缀。

结构：

```js
+字符串
```

> **+作用总结**
>
> 表示正数符号： +10
>
> 字符串拼接符号：'abc' + 'efg'
>
> 数字加法符号：10 + 10
>
> 数字的隐形转换：+'123'

## 二、运算符

作用：发起运算的最简单形式

分类：

数学运算符

比较运算符

逻辑运算符

赋值运算符

按位运算符

条件(三元运算符)

### 2.1.数学运算符

| 运算符 | 含义       |
| ------ | ---------- |
| +      | 加法运算   |
| -      | 减法运算   |
| *      | 乘法运算   |
| /      | 除法运算   |
| %      | 取余数运算 |
| ()     | 优先运算   |

运算顺序：先算乘除取余，再算加减，遇见小括号先算小括号

特殊运算：纯数字字符串与数字进行运算，纯数字字符串会被隐式转换成数字，参与运算(加法除外，字符串拼接优先级高于数字加法)，纯数字之外，还有布尔值和null会被隐式转换

隐式转换：不用parseInt和parseFloat，自动将字符串转成数字

如：true-->1 false-->0  null-->0  "123"--> 123

undefined，或者非纯数字字符串不会被隐式转换，得到结果位NaN

**总结:**

> 1.纯数字字符串与数字进行除加法外的数学运算时，都能进行隐式转换
>
> 2.特殊字符布尔值，null与数字进行计算时，所有的数学运算中都能被隐式转换
>
> 3.非纯数字字符串和undefined，不能进行隐式转换，与任何数字进行运算得到的都是NaN
>
> 4.任何数字和NaN进行计算都会得到NaN

**注意:**

> 浮点数计算精确的度有问题那是因为所有数据都是二进制形式存储
>
> 例如：-5.5
>
> 二进制表示：
>
> 符号位：1表示负数，0表示正数
>
> 整数位：依次是2^n...2^0
>
> 小数位：依次是2^-1...2^-n
>
> 解决方法：转成整数在整除解决

### 2.2.比较运算符

| 运算符 | 含义                                                         |
| ------ | ------------------------------------------------------------ |
| >      | 大于                                                         |
| <      | 小于                                                         |
| >=     | 大于等于                                                     |
| <=     | 小于等于                                                     |
| ==     | 等于，判断两边关系的时候尽可能去判断两边相等                 |
| !=     | 不等于                                                       |
| ===    | 全等于，判断两边关系的时候尽可能去判断两边不相等，除了判断值还会判断数据类型 |
| !===   | 不全等于                                                     |

所有比较运算符参与运算得到的结果都是会返回布尔值

**特殊情况：**

**特殊字符串与数字进行比较：**

纯数字字符串会隐式转换成数字，true转为1，false转为0，null转为0，但是做相等相等和全等判断的时候，null不等于0

**字符串与字符串进行比较：**比较的不是大小、长短，比较的是字符的Unicode编码顺序。顺序靠前的小于顺序靠后的，如果第一个字符相同依次往后面进行判断，直到判断除大小

**字符编码顺序：**数字 < 大写字母 < 小写字母

NaN参与的情况：NaN与任何比较都是false

Infinity参与的情况：+=Infinity是不相同的

### 2.3.逻辑运算符

逻辑运算符常用于布尔类型值之间；当操作数都是布尔值时，返回值也是布尔值

| 运算符 | 含义                                                       |
| ------ | ---------------------------------------------------------- |
| &&     | 逻辑与运算符，比较两个值只有两个值都为真才为真，否则为假   |
| \|\|   | 逻辑或运算符，比较两个值只有任意一个值为真就为真，否则为假 |
| !      | 逻辑非运算符，取反                                         |

**特殊情况：**

其他非布尔值也可以参与逻辑运算，参与的时候，会自动转换成布尔值进行运算，返回的结果就不一定是布尔值。

NaN、0、null、""、undefined会转换成false参与运算

非0数字、非空字符串都会转成true参与运算

> 注意：
>
> !!可以将任何数据转换成布尔值
>
> NaN、0、null、""、undefined转成布尔值后都是false其余都是true

### 2.4.赋值运算符

赋值运算符是赋一个基于它右边操作数值的值给它左边的操作数。最简单的赋值运算符是(=),将右边的操作数值付给左边的操作数。

作用：将右侧的数据运算后赋值给左边

| 运算符 | 含义                                                     |
| ------ | -------------------------------------------------------- |
| =      | 直接赋值，将等号右侧值赋值给左侧变量                     |
| +=     | 加等于(在变量原基础上，进行右侧运算，再将运算值给左边)   |
| -=     | 减等于(在变量原基础上，进行右侧运算，再将运算值给左边)   |
| *=     | 乘等于(在变量原基础上，进行右侧运算，再将运算值给左边)   |
| /=     | 除等于(在变量原基础上，进行右侧运算，再将运算值给左边)   |
| %=     | 取余等于(在变量原基础上，进行右侧运算，再将运算值给左边) |
| ++     | 递加(在变量原基础上，进行右侧运算，再将运算值给左边)     |
| --     | 递减(在变量原基础上，进行右侧运算，再将运算值给左边)     |

> 注意：
>
> ++或--写在前面，会先运算，后返回变量
>
> ++或--写在后面，会先返回变量，后运算

### 2.5.三元运算符

作用：符合条件赋值，三元运算符是为一个需要三个元素参与的表达式

结构：

```js
变量 = 条件表达式?值1:值2;
// 当表达式为真时，取值1，否则取值2
```

### 2.6.位运算符

作用：针对二进制的运算符

| 运算符 | 含义                         |
| ------ | ---------------------------- |
| &      | 与，都为1才为1               |
| \|     | 或，有1则为1                 |
| ~      | 非，所有取反                 |
| ^      | 异或，相同为0不同为1         |
| <<     | 零填充左位移，整体左移多少位 |
| >>     | 有符号右位移                 |
| >>>    | 零填充右位移                 |

注：负数是以补码的形式，js采取32位来存储数据

例如：如果一个8位的有符号整数，这个二进制数应该表示为：00001101，最高一位是符号位。负数的话，先置符号为1，剩下的位按位取反，末位加1，最后的结果是11110011。
